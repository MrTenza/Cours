• npm init -y
  • Ensuite créer 3 fichier : babel.config.js , es6.js , index.html
  • Dans le terminale recuperer les package de babel : npm i @babel/core @babel/cli @babel/preset-env
  • Pour lancer un script present dans package.json: npm run "le nom du script"
  • Pour qu'un nouveau fichier soit créer apres le lancement du script on peut ajouter dans package.json , dans les script : babel es6.js -o es6after.js
  • Pour configurer babel.config.js faire : module.exports = { presets: [["@babel/preset-env"]], };
